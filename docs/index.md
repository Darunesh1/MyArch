# MyArch Documentation

MyArch is a comprehensive configuration repository for **Hyprland**, a dynamic tiling Wayland compositor on Arch Linux. This documentation covers the setup, configuration, and usage of all included components.

## Components

| Component | Description |
|-----------|-------------|
| [**Hyprland**]({{ site.baseurl }}/hyprland/) | Window manager configuration with animations, keybindings, and workspace rules |
| [**Waybar**](#) | Customizable status bar with modules for system monitoring |
| [**Kitty**](#) | GPU-accelerated terminal with custom layouts and keybindings |

## Features

- **Animated Transitions**: Smooth window and workspace animations with bezier curves
- **Dwindle Layout**: Efficient automatic tiling with pseudotiling support
- **Custom Status Bar**: Waybar with One Dark theme and system modules
- **Wallpaper Cycling**: Automatic wallpaper rotation with smooth transitions
- **Clipboard Management**: Integrated cliphist with rofi menu
- **Screenshot Tools**: Grim + slurp for area and fullscreen captures

## Requirements

- Arch Linux (or Arch-based distribution)
- Wayland-compatible hardware
- GPU with proper drivers (NVIDIA, AMD, or Intel)
